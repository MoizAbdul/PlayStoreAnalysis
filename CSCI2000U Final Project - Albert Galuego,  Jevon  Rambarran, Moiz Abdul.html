Showing paid to free ratio using pie api¶

  • use the pie api to graph the ratio between paid and free apps in Game, Family, Tools, and Medical.
  • an outside ring is used to plot the percentage that each category takes up in the pie graph
  • an inside ring is used to illustrate the ratio between paid and free apps in each respective category
In [24]:
new_data = data.groupby(['Category', 'Type']).agg({'App' : 'count'}).reset_index()

outer_group_names = ['GAME', 'FAMILY', 'MEDICAL', 'TOOLS']
outer_group_values = [len(data.App[data.Category == category]) for category in outer_group_names]

a, b, c, d=[pl.cm.Blues, pl.cm.Reds, pl.cm.Greens, pl.cm.Purples]

inner_group_names = ['Paid', 'Free'] * 4
inner_group_values = []

for category in outer_group_names:
    for t in ['Paid', 'Free']:
        x = new_data[new_data.Category == category]
        try:
            inner_group_values.append(int(x.App[x.Type == t].values[0]))
        except:
            inner_group_values.append(0)

explode = (0.025, 0.025, 0.025, 0.025)
# outside ring
fig, ax = pl.subplots(figsize=(10, 10))
ax.axis('equal')
mypie, texts, _ = ax.pie(outer_group_values, radius=1.2, labels=outer_group_names, autopct='%1.1f%%', pctdistance = 1.1,
                                 labeldistance= 0.75,  explode = explode, colors=[a(0.6), b(0.6), c(0.6), d(0.6)], textprops={'fontsize': 16})
pl.setp( mypie, width=0.5, edgecolor='black')
 
# inside ring
mypie2, _ = ax.pie(inner_group_values, radius=1.2-0.5, labels=inner_group_names, labeldistance= 0.7, 
                   textprops={'fontsize': 12}, colors = [a(0.4), a(0.2), b(0.4), b(0.2), c(0.4), c(0.2), d(0.4), d(0.2)])
pl.setp( mypie2, width=0.5, edgecolor='black')
pl.margins(0,0)
 
# display graph
pl.tight_layout()
pl.show()
  • The majority of the top categories of apps are free download favoured
